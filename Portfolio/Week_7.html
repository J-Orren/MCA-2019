This week, I created a CSS file so as to display the metadata I had implemented within my original MEI file in a clear, concise and helpful manner to any users of my music collection.

You can find the link to this page (with metadata attached) below, which would be used as the main viewing page for all compositions in my collection.

You can find a link to my CSS file below as well.


Metadata presented under a Blue heading refers to who composed the original work, when it was created, and which game it was created for. It also denotes who originally transcribed it for the collection.

Metadata presented under a Red heading refers to who's added to the original transcription, when they added to it, and which draft of the transcription it is.

Metadata presented under a Green heading refers to what type of genre of the game the music appears in.
